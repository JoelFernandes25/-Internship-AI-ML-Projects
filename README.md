# Internship Portfolio: AI/ML Engineering at Labmentix

This repository contains the core technical projects completed during my internship, focusing on **Data Science**, **Machine Learning**, and **Operations Research**.

## 🛠 Project Modules

### 1. Retail Store Forecasting & Optimization
* **Goal:** Predict retail demand and optimize store operations using time-series forecasting.
* **Key Files:** `retail_store_forecasting_and_optimization.ipynb`
* **Skills:** Time-series analysis, Optimization algorithms.

### 2. Customer Satisfaction (CSAT) Modeling
* **Goal:** Predict customer satisfaction levels and analyze feedback loops.
* **Key Files:** `CSAT Prediction.ipynb`, `DeepCSAT.ipynb`
* **Skills:** Classification, Deep Learning (Neural Networks).

### 3. Fitness Data Engineering
* **Goal:** Visualize athletic performance and manage data connections.
* **Key Files:** `Strava Data Viz.ipynb`, `Strava SQL data connection.ipynb`
* **Skills:** Data Visualization, SQL Integration, PowerBI

### 4. Market Research & Financial Analysis
* **Goal:** Analyze sentiment and financial trends for major banking and media entities.
* **Key Files:** `Yes Bank Project.ipynb`, `IMDB Project.ipynb`
* **Skills:** Exploratory Data Analysis (EDA), Market Sentiment.
